### notes
#### use case
> this is not meant to be used by anyone else but you can use it as a **reference**\
> *flakes* are a new way to manage nix packages and configurations\
> and `flake-parts` is a library that helps you manage your *flakes* in a more modular way

#### nixos and nix store
> nixos is an operating system based off nix package manager\
> which uses `declarative` configuration instead of traditional\
> imperative installation

> and it works using nix store with usage of symlinks or\
> wrappers which make programs work
